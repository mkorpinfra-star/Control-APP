Sobre o Sistema

PuntoTouch é uma plataforma completa de gestão de obras com arquitetura multi-tenant, permitindo que múltiplas empresas utilizem o mesmo sistema com isolamento total de dados.

O sistema oferece controle completo de obras, clientes, funcionários, encarregados, administradores, além de módulos financeiros (faturamento, folha de pagamento), apontamento de horas, aprovações e analytics.

Novidades da Versão 2.0

Recursos Recém-Implementados

  • Monitoramento Real-Time: Visualização ao vivo de funcionários que não marcaram ponto
  • Notificações Push via Firebase: Mensagens instantâneas no celular via FCM API V1
  • Broadcast de Avisos: Envio de notificações para TODOS os usuários
  • Integração WhatsApp Aprimorada: Mensagens com informações da obra
  • Sistema Dual de Notificações: Sistema separado para atividades e monitoramento
