NOVO DESIGN PARA TIMESHEET - ESTILO NUBANK MODERNO

Em vez da tabela complexa com inputs pequenos, criar:

1. **Cards por dia** (vertical, um embaixo do outro)
   - Cada dia é um card `bg-[#F5F5F5] rounded-2xl`
   - Header do card: Nome do dia + data pequena
   - Botão festivo no canto superior direito (toggle)

2. **Sistema de input moderno com botões rápidos**
   Para cada tipo de hora (Normal, Extra, Noturna):
   - Botões rápidos: +0.5h, +1h, +2h, +4h, +8h
   - Display grande central mostrando o total
   - Botão reset/limpar
   - Input manual pequeno se necessário ajuste fino

3. **Visual clean**
   - Cores sutis: verde para Normal, laranja para Extra, roxo para Noturna
   - Animações suaves ao adicionar horas
   - Feedback visual imediato

4. **Total card no final**
   - Card destacado em cinza escuro mostrando totais
   - Animação ao atualizar valores

5. **Botões de ação**
   - Salvar e Enviar em botões grandes rounded-xl
   - Estado desabilitado visual claro

Este design reduz de ~10 toques para marcar um dia completo para ~3 toques.
Exemplo: Segunda normal 8h = toque no card Segunda + toque no botão +8h = pronto!
